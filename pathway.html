Data Collection of Vascular Procedures

Patient Admission / Pathway
Data elements collected -
  1. Date of Admission
  2. Mode of Admission - "Elective";"Non-Elective"
  3. Date of Referral
  4. Referring Specialty - "Primary Care";"Hospital Department";"Other Source"

    5. Selecting "Hospital Department" allows a selection form the end user editable table currently storing the values -
    "Cardiac Services", "Cardiology", "ED", "Emergency Medicine/Trauma Services", "Ophthalmology", "Radiology"

CEA admissions add 'CEA Status on referral' with values "Asymptomatic";"Symptomatic". Prior history on symptomatic CEA referrals are stored in the popup form 'CEA history on referral'
